# appelSet

Создайте метод с именем checkMultipleElements, который принимает:

HashSet строк (set)
ArrayList строк (elements) для проверки
Для каждого элемента в списке выведите true, если элемент существует в множестве, в противном случае выведите false.

Пример ввода:

set = ["Apple", "Banana", "Cherry"]
elements = ["Banana", "Grapes", "Apple"]

Ожидаемый вывод:

true
false
true
